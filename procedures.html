The split

From long SOPs to focused procedures

As we migrate each SOP, we separate reference from procedure — definitions, safety, equipment, layout and roles are shared knowledge, not tasks — then tag every procedure by job function. Some belong to two roles (Clamp Unload = Operator + Dock Associate), so the tag is a set.
